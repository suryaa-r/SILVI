# SILVI Technology Stack

## Programming Languages
- **HTML5**: Semantic markup with modern web standards
- **CSS3**: Advanced styling with custom properties and animations
- **JavaScript (ES6+)**: Modern JavaScript for interactive functionality

## Frontend Framework & Libraries
- **Tailwind CSS v3**: Utility-first CSS framework for responsive design
- **Anime.js v3.2.1**: Lightweight animation library for smooth effects
- **Typed.js v2.0.12**: Typewriter effect library for dynamic text
- **Splide.js v4.1.4**: Touch-friendly carousel/slider library

## External Services & APIs
- **Razorpay Checkout v1**: Payment gateway integration for secure transactions
- **Google Fonts**: Premium typography (Playfair Display, Inter)
- **CDN Resources**: External hosting for libraries and dependencies

## Development Environment
- **No Build System**: Direct HTML/CSS/JS development approach
- **CDN Dependencies**: External library loading via CDN links
- **Static Hosting Ready**: Deployable to any static web hosting service

## Browser Compatibility
- **Modern Browsers**: Chrome, Firefox, Safari, Edge (ES6+ support required)
- **Mobile Responsive**: Touch-friendly interface with responsive breakpoints
- **Progressive Enhancement**: Graceful degradation for older browsers

## File Structure & Organization
- **Modular CSS**: Component-based styling with CSS custom properties
- **Separated Concerns**: HTML structure, CSS presentation, JS behavior
- **Asset Management**: External image hosting via CDN services

## Performance Optimizations
- **Lazy Loading**: Deferred loading of non-critical resources
- **CSS Animations**: Hardware-accelerated transitions and effects
- **Optimized Images**: External CDN hosting for faster loading
- **Minimal Dependencies**: Lightweight library selection for fast performance

## Development Commands
Since this is a static HTML/CSS/JS project:
- **Local Development**: Open index.html in browser or use local server
- **Live Server**: Use VS Code Live Server extension for development
- **File Watching**: Browser auto-refresh on file changes (with live server)
- **Deployment**: Upload files to static hosting (Netlify, Vercel, GitHub Pages)

## Security Considerations
- **Payment Security**: Razorpay handles sensitive payment data
- **HTTPS Required**: Secure connection needed for payment processing
- **Input Validation**: Client-side validation with server-side backup recommended
- **Content Security**: External CDN resources from trusted providers
